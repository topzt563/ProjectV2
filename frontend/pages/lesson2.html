บทเรียนที่ 2 (Chapter 2)

เตรียมความพร้อม
ติดตั้ง Git & สมัคร GitHub

ก่อนเริ่มเขียนโค้ด เราต้องมีเครื่องมือสำหรับจัดการไฟล์ในคอมพิวเตอร์ (Git) และพื้นที่สำหรับฝากไฟล์บนคลาวด์ (GitHub) กันก่อน

ส่วนที่ 1: การติดตั้ง Git ลงในเครื่อง

1

ดาวน์โหลดและติดตั้ง

เข้าไปที่เว็บไซต์ git-scm.com เลือกดาวน์โหลดตามระบบปฏิบัติการของคุณ (Windows, Mac, Linux) แล้วกดติดตั้งแบบ Next ไปเรื่อยๆ (แนะนำให้ใช้ Git Bash สำหรับ Windows)

2

ตั้งค่าตัวตน (Config Identity)

เปิด Terminal (หรือ Git Bash) แล้วพิมพ์คำสั่งด้านล่างเพื่อบอกให้ Git รู้ว่าใครเป็นคนเซฟงาน (ทำแค่ครั้งแรกครั้งเดียว)

# ตั้งชื่อของคุณ (ภาษาอังกฤษ)
git config --global user.name "Your Name"
# ตั้งอีเมลของคุณ
git config --global user.email "your.email@example.com"

ตัวอย่างภาพหน้าจอการใช้งานโค้ด

ส่วนที่ 2: การสมัครบัญชี GitHub

1

ไปยังเว็บไซต์ GitHub

เปิดเบราว์เซอร์ไปที่ github.com จากนั้นคลิกที่ปุ่ม "Sign up" ที่มุมขวาบนของหน้าจอ

2

กรอกข้อมูลการสมัคร

ระบบจะให้กรอก อีเมล (แนะนำให้ใช้อีเมลเดียวกับที่ตั้งใน Git Config ด้านบน), รหัสผ่าน และ ยูสเซอร์เนม (Username)

3

ยืนยันอีเมล (Verify Email)

เข้าไปเช็คในกล่องจดหมายของคุณ เพื่อกดยืนยันการสมัคร เพียงเท่านี้คุณก็พร้อมอัปโหลดโค้ดขึ้นคลาวด์แล้ว!
