Version Control

เรียนรู้การใช้งาน Git และ GitHub อย่างเป็นขั้นตอน เพื่อจัดการไฟล์โค้ดอย่างปลอดภัย
และเพิ่มประสิทธิภาพในการทำงานร่วมกันเป็นทีมอย่างมืออาชีพ

เข้าสู่บทเรียน

โครงสร้างเนื้อหา (Course Outline)

เรียงลำดับเนื้อหาจากพื้นฐานสู่การใช้งานจริงในระดับองค์กร

1. พื้นฐานระบบควบคุมเวอร์ชัน

ข้อจำกัดของการจัดการไฟล์แบบดั้งเดิม และการเริ่มต้นใช้งาน Git สำหรับบันทึกประวัติการเปลี่ยนแปลง (Init, Add, Commit)

2. การบริหารจัดการสาขา (Branching)

หลักการแยกพื้นที่พัฒนา (Branch) เพื่อลดข้อผิดพลาด และแนวทางการปฏิบัติงานร่วมกันเป็นทีมอย่างปลอดภัย

3. การเชื่อมต่อคลังข้อมูลทางไกล

ปฏิบัติการฝากโปรเจกต์ไว้บนระบบคลาวด์ (Remote Repository) ผ่าน GitHub และการซิงโครไนซ์ข้อมูล (Push, Pull, Clone)

4. การจัดการข้อขัดแย้งขั้นสูง

การแก้ไขปัญหาเมื่อไฟล์มีการทับซ้อนกัน (Merge Conflict) และเทคนิคการย้อนสถานะไฟล์โค้ด (Reset / Revert)

คู่มือคำสั่ง Git (Command Guide)

รวมคำสั่ง Git พื้นฐานที่จำเป็นต้องใช้ในการทำงานจริง พร้อมคำอธิบายแบบกระชับ สำหรับใช้เป็นแหล่งอ้างอิงระหว่างการเขียนโปรแกรม
