บทที่ 2 – ติดตั้ง Git & สมัคร GitHub

เรียนรู้วิธีติดตั้ง Git บนทุกระบบปฏิบัติการ และสร้างบัญชี GitHub พร้อม config ตั้งต้น

▶ วิดีโอสาธิต – ติดตั้ง Git และ Config เบื้องต้น

ดาวน์โหลดและติดตั้ง Git

ไปที่เว็บไซต์ git-scm.com แล้วดาวน์โหลดเวอร์ชันที่เหมาะกับระบบปฏิบัติการของคุณ

🪟
Windows
ดาวน์โหลด .exe installer จาก git-scm.com แล้วติดตั้งแบบ Next → Next → Finish
🍎
macOS
ใช้ Homebrew: brew install git หรือติดตั้งผ่าน Xcode Command Line Tools
🐧
Linux (Ubuntu)
ใช้ apt: sudo apt install git

ตรวจสอบการติดตั้ง

bash – ตรวจสอบเวอร์ชัน Git
git --version # ผลลัพธ์ที่ควรได้ (เวอร์ชันอาจแตกต่างกัน) git version 2.44.0

Config ข้อมูลผู้ใช้ (ต้องทำก่อนใช้ครั้งแรก)

Git จะใช้ข้อมูลนี้เพื่อบันทึกว่าใครเป็นคนทำ Commit ต้องตั้งค่าครั้งเดียวต่อเครื่อง

bash – ตั้งค่าชื่อและอีเมล
# ตั้งชื่อผู้ใช้ (ใช้ชื่อจริงหรือชื่อ GitHub ก็ได้) git config --global user.name "ชื่อของคุณ" # ตั้งอีเมล (ใช้อีเมลเดียวกับ GitHub) git config --global user.email "email@example.com" # ตรวจสอบการตั้งค่าทั้งหมด git config --list

สร้างบัญชี GitHub

  1. ไปที่ github.com
    เปิดเบราว์เซอร์แล้วไปที่ github.com คลิกปุ่ม "Sign up"
  2. กรอกข้อมูล
    ใส่ Email, Password และ Username ที่ต้องการ ชื่อ Username จะใช้เป็น URL โปรไฟล์ของคุณ
  3. ยืนยันอีเมล
    GitHub จะส่ง code ไปที่อีเมล กรอก code เพื่อยืนยันตัวตน
  4. ตั้งค่าโปรไฟล์
    เพิ่มรูปโปรไฟล์ ชื่อ และ bio เพื่อให้โปรไฟล์ดูน่าเชื่อถือ
💡 เคล็ดลับ
ใช้อีเมลเดียวกันใน Git config และ GitHub account เพื่อให้ Commit ของคุณแสดงรูปโปรไฟล์ถูกต้อง
